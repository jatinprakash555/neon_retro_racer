Neon Racer: Ultimate is a browser-based 3D arcade racer where players speed through an infinite, procedurally generated neon city. The game features a custom physics engine for acceleration and drifting, dynamic difficulty scaling, and a fully procedural soundtrack generated in real-time using the Web Audio API (no external audio files required).

âœ¨ Features
-3D Graphics Engine: Built with Three.js featuring dynamic lighting, fog, bloom-style aesthetics, and shadow casting.
-Procedural Generation: Endless road, city skylines, traffic patterns, and weather effects (Neon Rain).
-Reactive Audio System: A custom SynthAudio engine that uses the Web Audio API to generate engine drones that pitch-shift with speed, beat synchronization, and sound effects without loading mp3/wav assets.
-Dynamic Gameplay:
	-Manual Acceleration & Braking.
	-Nitrous Boost system (speeds up to 500 km/h).
	-Power-ups: Shield (Invincibility) and Time Warp (Slow Motion).
	-Responsive Design: Fully optimized for both Desktop and Mobile devices.
	-Zero Dependencies: Everything is contained within a single .html file (libraries loaded via CDN).
